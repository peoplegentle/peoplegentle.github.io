# 나의 학습 여정

- **Entry**  
  블록 코딩을 통해 기본 개념과 논리적 사고를 익혔습니다.

- **Scratch**  
  다양한 창의적인 게임을 만들면서 코딩의 재미를 경험했습니다.

- **Python**  
  <i class="fa-brands fa-python"></i>
  텍스트 기반 프로그래밍을 배우고, 실제 응용 프로그램 개발에 도전하였습니다.
